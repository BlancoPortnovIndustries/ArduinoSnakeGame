~Русская версия README~

При запуске устройства появляется логотип игры. Он висит до тех пор, пока пользователь не нажмет на джойстик. 
Далее появляется надпись «SNAKE» и начинается выбор режима и сложности. Пользователю предлагается выбрать один из трех режимов: режим без стенок, «классический» режим (со стенками), зеркальная змейка. 
Для выбора режима необходимо передвинуть джойстик вверх или вниз. Одновременно с режимом игры следует выбор скорости – необходимо передвинуть джойстик вправо или влево. 
Первая скорость является минимальной, четвертая – максимальной. Для начала игры следует нажать 1 раз на джойстик. На матрице появляются 2 точки. 
Одна из них – мигающая – соответствует фрукту, вторая соответствует голове змеи. Для того, чтобы привести змейку в движение, необходимо двинуть джойстик в одну из четырех сторон.
Если змея врезается в саму себя или же в стенку (в зависимости от режима игры), игра заканчивается, появляется бегущая строка «GAME OVER» и на матрице зажигается количество очков, набранных за сеанс.
Если пользователь набрал 64 очка в первом или во втором режиме (максимальное число), то на дисплее матрицы выводится бегущая строка «YOU WIN». Эту же надпись можно получить на 3 режиме, набрав 32 очка.


~English version of README~

When the device starts, the game logo appears. It hangs until the user presses the joystick.
Then the inscription "SNAKE" appears and the selection of the mode and difficulty begins. The user is offered to choose one of three modes: mode without walls, "classic" mode (with walls), mirror snake.
To select a mode, move the joystick up or down. Simultaneously with the game mode, the choice of speed follows - you need to move the joystick to the right or left.
The first speed is minimum, the fourth is maximum. To start the game, press the joystick 1 time. 2 points appear on the matrix.
One of them - blinking - corresponds to the fruit, the second corresponds to the head of the snake. In order to set the snake in motion, you need to move the joystick in one of four directions.
If the snake crashes into itself or into the wall (depending on the game mode), the game ends, the scrolling line “GAME OVER” appears and the number of points scored for the session lights up on the matrix.
If the user scored 64 points in the first or in the second mode (maximum number), then the scrolling line “YOU WIN” is displayed on the matrix display. The same inscription can be obtained in mode 3 by gaining 32 points.

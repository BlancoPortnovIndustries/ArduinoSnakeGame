~РУССКАЯ ВЕРСИЯ README~

В данном разделе вы можете просмотреть / скачать:
	• файлы для печати корпуса в формате STL с различными допустимыми отклонениями по размерам
	• исходные файлы моделей для программы "Компас-3D"
	• эскиз, использованный для создания моделей, а также картинки с размерами устройств ввода/вывода и самой платы микроконтроллера
	• datasheet для матрицы 8x8 с ее размерами и схемой подключения светодиодов к выводам

~ENGLISH VERSION OF README~

In this section you can view / download:
	• files for printing the body in STL format with different size tolerances
	• source files of models for the Compass-3D program
	• the sketch used to create the models, as well as pictures with the dimensions of the input / output devices and the microcontroller board itself
	• datasheet for an 8x8 matrix with its dimensions and a diagram for connecting LEDs to the terminals
